Problem 1

Does an oriented graph on 101010 vertices with 100100100 edges exist? If yes, then is it necessarily Euler's?

Let's count the maximum number of vertices in an oriented graph. Since an incidence matrix for such a graph is a simple 10×1010\times1010×10 matrix with no restictions or symmetries, there are 10×10=10010\times10=10010×10=100 possible edges, which is precisely what we're being asked about.

This graph is certainly Euler's because all paths of length 101010 are cycles as follows, regardless of order (since the graph is perfectly symmetric and automorphic):

1→2→3→4→5→6→7→8→9→9→10→11\to2\to3\to4\to5\to6\to7\to8\to9\to9\to10\to11→2→3→4→5→6→7→8→9→9→10→1

Answer: yes, it exists, and yes, it is Euler's.
