Problem 1

Choose an arbitary function f ⁣:{1,2,…,n}↦{1,2,…,n}f\colon\{1,2,\dots,n\}\mapsto\{1,2,\dots,n\}f:{1,2,…,n}↦{1,2,…,n}. All outcomes are equally probable. Are events "f(1)=f(2)f(1)=f(2)f(1)=f(2)" and "f(2)=f(3)f(2)=f(3)f(2)=f(3)" independent?

Technically this might be sufficient since it's kinda obvious, but obviousness here frequently doesn't work, so I'll also present a stricter approach. Values f(n)f(n)f(n) are independent from one another, and every value of f(n)f(n)f(n) for any nnn is equally probable with a probability of 1n\displaystyle\frac{1}{n}n1​. Let's fix the value of f(2)=constf(2)=\text{const}f(2)=const. Then, the first event is dependent only on the value of f(1)f(1)f(1), and the second event is dependent only on the value of f(3)f(3)f(3), which means they're independent.

For the events to be independent, the following has to be true:

P[A∩B]=P[B]⋅P[A]P[A\cap B]=P[B]\cdot P[A]P[A∩B]=P[B]⋅P[A]

What is the probability of P[B]P[B]P[B] and P[A]P[A]P[A] independently? Kinda follow the approach above. Say, that we have already chosen f(2)f(2)f(2), and need to choose f(1)f(1)f(1) for event AAA and f(3)f(3)f(3) for event BBB. For the events to happen, our chosen values for f(1),f(3)f(1), f(3)f(1),f(3) have to be the same as f(2)f(2)f(2). f(2)f(2)f(2) has already been chosen out of nnn equally probable options, so the chance of choosing the same option as f(2)f(2)f(2) for each of these cases (independently, for now) is 1n\displaystyle\frac{1}{n}n1​.

Note that the above explanations apply symmetrically if we were to first choose f(1)f(1)f(1) or f(3)f(3)f(3), respectively, and only then choose f(2)f(2)f(2) for each of the cases.

Thus,

P[A]=1n,P[B]=1nP[A]=\frac{1}{n},\quad P[B]=\frac{1}{n}P[A]=n1​,P[B]=n1​

Now, what are the chances of the event A∩BA\cap BA∩B occurring? This event would be called "f(1)=f(2)=f(3)f(1)=f(2)=f(3)f(1)=f(2)=f(3)". Let's calculate its chance of occurring: fix f(1)f(1)f(1) to some value. We need to calculate the chance that f(2),f(3)f(2),f(3)f(2),f(3) would be the same as f(1)f(1)f(1), so we need to choose a specific number out of nnn options, twice, thus giving us a probability of 1n⋅1n=1n2\displaystyle\frac{1}{n}\cdot\frac{1}{n}=\frac{1}{n^2}n1​⋅n1​=n21​.

Thus,

P[A∩B]=1n2=1n⋅1n=P[A]⋅P[B]P[A\cap B]=\frac{1}{n^2}=\frac{1}{n}\cdot\frac{1}{n}=P[A]\cdot P[B]P[A∩B]=n21​=n1​⋅n1​=P[A]⋅P[B]

which means that the events are independent from each other.

Problem 2

In a lotto, four numbers out of {1,2,…,16}\{1,2,\dots,16\}{1,2,…,16} are chosen randomly and with equally probable chances. Find the probability of an event AAA "there is no 131313 among the chosen numbers" under a condition BBB that "there is no 111 among the chosen numbers".

The probability space is all sequences with non-repeating numbers of length 444 from the set above. This gives us

A164=16!(16−4)!=16×15×14×13A^{4}_{16}=\frac{16!}{(16-4)!}=16\times15\times14\times13A164​=(16−4)!16!​=16×15×14×13

total possible sequences.

The number of sequences when event AAA or event BBB occur are (we just remove a single number from the pool of possible options):

A154=15!(15−4)!=15×14×13×12A^4_{15}=\frac{15!}{(15-4)!}=15\times14\times13\times12A154​=(15−4)!15!​=15×14×13×12

Now, we know that

P[A∣B]=P[A∩B]P[B]P[A|B]=\frac{P[A\cap B]}{P[B]}P[A∣B]=P[B]P[A∩B]​

Event A∩BA\cap BA∩B could be defined as "there is neither 131313 or 111 among the chosen numbers", which gives us

A144=14!(14−4)!=14×13×12×11A^4_{14}=\frac{14!}{(14-4)!}=14\times13\times12\times11A144​=(14−4)!14!​=14×13×12×11

possible options.

Now,

P[B]=15×14×13×1216×15×14×13=34P[B]=\frac{15\times14\times13\times12}{16\times15\times14\times13}=\frac{3}{4}P[B]=16×15×14×1315×14×13×12​=43​

P[A∩B]=14×13×12×1116×15×14×13=34⋅1115=3360P[A\cap B]=\frac{14\times13\times12\times11}{16\times15\times14\times13}=\frac{3}{4}\cdot\frac{11}{15}=\frac{33}{60}P[A∩B]=16×15×14×1314×13×12×11​=43​⋅1511​=6033​

Finally,

P[A∣B]=336034=1115P[A|B]=\frac{\frac{33}{60}}{\frac{3}{4}}=\frac{11}{15}P[A∣B]=43​6033​​=1511​

which is the final answer.

Problem 4

In the first box there are 999 chips enumerated from 111 to 999. In the second box there are 101010 chips, enumerated from 222 to 111111. Chips in each box differ from each other only by numbers. A random box is chosen with equal probabilities, and then a random chip is chosen with equal probabilities from the chosen box. What is the probability that a box with 101010 chips is chosen (event AAA) if a chip with number 777 was pulled out (event BBB)?

Let's denote chosen chips with two coordinates: its box number and its own number. Formula from above strikes again:

P[A∣B]=P[A∩B]P[B]P[A|B]=\frac{P[A\cap B]}{P[B]}P[A∣B]=P[B]P[A∩B]​

The chance that we choose either boxes is 12\displaystyle\frac{1}{2}21​. Thus, we may split the field of probabilities into two parts. The first part would have each of the events "chips with certain coordinates is pulled"

(1,1),(1,2),…(1,9)(1, 1), (1, 2), \dots (1, 9)(1,1),(1,2),…(1,9)

occur with a probability of 12×19=118\displaystyle\frac{1}{2}\times\frac{1}{9}=\frac{1}{18}21​×91​=181​ since we divide this half into 999 equally probable events.

Whereas similar events for the second box

(2,2),(2,3),…(2,11)(2, 2), (2, 3), \dots (2, 11)(2,2),(2,3),…(2,11)

each occur with a probability of 12×110=120\displaystyle\frac{1}{2}\times\frac{1}{10}=\frac{1}{20}21​×101​=201​ since here we have 101010 equally probable options.

Thus, the chance of pulling a chip with number 777 is equal to the sum of respective probabilities from each of the boxes:

P[B]=118+120=19180P[B]=\frac{1}{18}+\frac{1}{20}=\frac{19}{180}P[B]=181​+201​=18019​

The chance of a certain chip with number 777 from the second box is simply

P[A∩B]=120P[A\cap B]=\frac{1}{20}P[A∩B]=201​

Finally,

P[A∣B]=12019180=919P[A|B]=\frac{\frac{1}{20}}{\frac{19}{180}}=\frac{9}{19}P[A∣B]=18019​201​​=199​

which is the final answer.
