Problem 1.2

Consider whole numbers x,y,z,t,wx, y, z, t, wx,y,z,t,w, and two statements: A=x+y+z+t+wA = x + y + z + t + wA=x+y+z+t+w is even, B=xyztwB = xyztwB=xyztw is even. Prove that A→BA \rightarrow BA→B is true.

Solution

AAA is true only if 0,20, 20,2, or 444 values are odd (sum of two odd numbers is even), otherwise it's odd. In other words, AAA is true only if there are precisely 5,35, 35,3, or 111 even numbers.

BBB is true if at least one value is even.

Consider the following expression:

"exactly 111 value is even" ∨\lor∨ "exactly 333 values are even" ∨\lor∨ "exactly 555 values are even" →\rightarrow→ "at least 111 value is even"

If either 1,31, 31,3, or 555 values are even, then the statement "at least 111 value is even" is always true, therefore A→BA \rightarrow BA→B is always true, q. e. d.
