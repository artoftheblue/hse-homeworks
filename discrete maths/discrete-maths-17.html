Problem 1

Consider partial orders, in which there are precisely 555 minimal and 555 maximal elements. Find the smallest number of elements in such partial orders.

Per given conditions, there should be 555 different minimal and 555 different maximal elements (if two minimal/maximal elements overlap, their number would decrease).

Per definitions:

  • an element is maximal if ∄x∈P\nexists x\in P∄x∈P such that x<ax < ax<a
  • an element is minimal if ∄x∈P\nexists x\in P∄x∈P such that x>ax > ax>a

Take 555 independent maximal elements and make them also minimal elements. This means that all of these elements are incomparable and thus are all both maximal and minimal. If we were to remove any of these elements, we would go below the required condition of having 555 elements   ⟹  \implies⟹ it's impossible to have a smaller number of elements in such partial orders.

Answer: 555.

Problem 3

Prove that a strict order on 141414 elements with 505050 adjacent pairs does not exist.

In order to disprove this, we should somehow find the maximum number of pairs in such a relation. For this, we shall take an element and make it comparable to some number of other elements.

We could split all elements into a bipartite graph in some proportion and try to maximize the number of total edges between the parts. The action of splitting all elements into two parts is justified over splitting the elements into n-partite graphs with n>2n>2n>2 since we want to maximize the number of edges (why would we create more parts if that would strip us of the possibility to make edges within the part?).

This is perhaps the most common optimization problem of maximizing the area of a rectangle with a certain perimeter (the maximum area is when the sides are the closest to each other in terms of length, thus the maximum number of edges is 7×7=497\times7=497×7=49, and not 505050), but I'll present a proper solution:

The number of elements in the left part of the graph is NNN, the number of elements in the right part of the graph is N−14N-14N−14. The number of total pairs is f(N)=N(14−N)f(N)=N(14-N)f(N)=N(14−N).
